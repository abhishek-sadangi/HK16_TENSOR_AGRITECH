# HK16_TENSOR_AGRITECH

🧠 Project Overview

This project is a Crop Price Prediction Web Application built using Streamlit (by Streamlit Inc). It uses a hybrid machine learning approach combining Prophet (time-series forecasting) and XGBoost (residual error correction) to predict agricultural commodity prices based on historical data.

🚀 Approach

Built an interactive web UI using Streamlit with futuristic styling.

Implemented session-based navigation (home → dashboard).

Loaded and processed CSV data; fallback synthetic data if file missing.

Time-series forecasting using Prophet.

Error correction using XGBoost for better accuracy.

Hybrid prediction = Prophet forecast + XGBoost residual.

Displayed results via charts and data tables.

Added CSS for modern UI/UX.

🛠 Technologies Used

Streamlit (frontend)

Prophet (time-series forecasting)

XGBoost (machine learning)

Pandas (data handling)

NumPy (numerical computation)

CSS (custom styling)

Python (core logic)

📌 Features

Crop price prediction

Historical data visualization

Hybrid ML forecasting

Futuristic UI

Error handling & fallback data

🎯 Purpose

To demonstrate a practical ML-powered agricultural prediction system with an interactive web interface.
